# 🌦️ Atmos Weather Dashboard

A modern and responsive weather dashboard application that provides real-time weather updates, temperature details, humidity, wind speed, and forecast information for cities around the world.

---

# 📌 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature updates
* 💧 Humidity information
* 💨 Wind speed details
* ☁️ Weather conditions with icons
* 📱 Fully responsive design
* 🌙 Modern UI design
* ⚡ Fast API integration

---

# 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js
* OpenWeather API

# 🚀 Installation & Setup

## Clone Repository

```bash
git clone https://github.com/ratneshk12/Atmos-Weather-Dashboard.git
```

## Open Project Folder

```bash
cd Atmos-Weather-Dashboard
```

## Install Dependencies

```bash
npm install
```

## Run Project

```bash
node server.js
```

---

# 🔑 API Setup

1. Create account on OpenWeather API
2. Generate your API key
3. Add API key inside project configuration

Example:

```javascript
const apiKey = "YOUR_API_KEY";
```
